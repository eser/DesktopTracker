# DesktopTracker

Simply allows you to start and stop tasks.

![Sey](docs/desktop-tracker.png)


### Features

- Global Hotkey: Win+Delete
- Task Hotkeys: Ctrl+N = New Task, Ctrl+Number = Start/Pause Task
